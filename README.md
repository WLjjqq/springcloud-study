什么是服务者？什么是消费者？
====================
服务者：为其他服务做服务的。
消费者：依赖于其它服务。
假如：用户去电影院买票。用户微服务这个模块要先去查询一下这个用户的余额是否充足。电影微服务这个模块根据用户微服务发送过来的信息决定是否卖给用户。
这里：用户微服务  是提供者。  电影微服务   是消费者。前提是提供者的服务器要打开。
